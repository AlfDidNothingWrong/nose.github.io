JiveX DICOM Viewer

JiveX [dv] ist ein mit modernsten Javatechnologien entwickeltes Bildvisualisierungs- und Bildbearbeitungswerkzeug für den Mediziner.

Sie können DICOM Dateien direkt von einem Speichermedium (wie z.B. von der Festplatte, aus einem Netzwerkverzeichnis oder von einer CD-ROM) öffnen und bearbeiten.

Auch Bilder die Sie z.B. per E-Mail empfangen oder über das Internet geladen haben können direkt visualisiert werden. Dateien mit der Dateiendung .dcm werden als DICOM Dateien erkannt. Dies ermöglicht Ihnen, DICOM Bilddateien mit dieser speziellen Dateiendung durch einen Doppelklick zu öffnen.

Zur Bildbearbeitung stehen Ihnen Standardfunktionen (wie z.B. die Grauwertfensterung auf "echten" DICOM Bilddaten, verschiedene Vergrößerungsfunktionen und die CINE-Funktion zur Animation) zur Verfügung. Des weiteren können Winkel- und Distanzanzeigen durchgeführt, sowie graphische und textuelle Annotationen in verschiedenen Farben gezeichnet werden. JiveX [dv] unterstützt bereits die neuen DICOM Softcopy Presentation States. Diese ermöglichen es den Zustand von DICOM Bildern wie z.B. die Fensterung, den Darstellungsbereich sowie Annotationen in einem herstellerunabhängigen Format zu speichern.

Als besondere Zusatzfunktion bietet Ihnen JiveX [dv] die Möglichkeit, im Vergleichsmodus zwei unterschiedliche Bildserien in jeweils einem eigenen Bildbereich parallel anzuzeigen, um diese dann miteinander vergleichen zu können. JiveX [dv] kann auf Farbmonitoren eine softwareseitige Korrektur der Bildanzeige vornehmen, so daß Monochrombilder gemäß des DICOM Standards angezeigt werden.

JiveX DICOM Viewer Light entspricht allen zutreffenden Anforderungen der Richtlinie 93/42/EWG und ist damit ein Medizinprodukt. Für die Nutzung der DICOM-Bilddaten im radiologischen Befundungsprozess ist die Röntgenverordnung zu beachten.

Wie alle JiveX Systeme wurde auch JiveX [dv] in reinem Java implementiert und ist daher plattformunabhängig einsetzbar. Auf fast allen Betriebssystemen ist JiveX [dv] mit gleichen Funktionen und gleicher Benutzeroberfläche verfügbar.

JiveX [dv] ist z.Zt. in folgenden Sprachen verfügbar: Englisch, Deutsch.

Empfohlene Systemausstattung

  • Java 1.6 (Java2) oder höher (zur Zeit verfügbar für Windows, Linux, MacOS X, Solaris und weitere UNIX Betriebssysteme)
  • Mindestens 256 MB Hauptspeicher
  • Grafiksystem mit Echtfarbendarstellung (TrueColor)

Installation

Dieser Patienten CD liegt die Windows-Version des JiveX DICOM Viewers bei.

Die Installation starten Sie indem Sie die Setup-Datei im Verzeichnis /Setup der CD ausführen.
